# 📥 Como copiar e instalar

## 1️⃣ Clone o repositório

```bash
git clone https://github.com/davizintech-ops/hatsune-bot.git
cd hatsune-bot
```

## 2️⃣ Instale as dependências

```bash
npm install
```

## 3️⃣ Configure o bot

Edite o arquivo **config.json** conforme necessário.

## 4️⃣ Inicie o bot

```bash
npm start
```

ou

```bash
node index.js
```

## 5️⃣ Conecte ao WhatsApp

Na primeira execução, siga as instruções exibidas no terminal para parear o bot com sua conta do WhatsApp.
